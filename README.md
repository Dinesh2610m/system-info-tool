# 🖥️ System Info Tool

A simple Python-based system information tool built and deployed on a cloud VM.

## 🚀 Features
- Displays current user
- Shows OS and system details
- Displays machine hostname
- Shows CPU information
- Displays disk usage (total, used, free)

## 🛠️ Tech Stack
- Python
- Linux (Ubuntu VM)
- Git & GitHub

## ☁️ Deployment
This project was built and executed on a cloud-based virtual machine.

## ▶️ How to Run

```bash
python3 app.py
